Test de l'enregistrement d'une question

Quand un usager pose la question "Quel temps fait-il à Bordeaux ?"
  • Alors le serveur indique qu'il à enregistré la question avec le code : 201
  • Et il permet à l'usager de localiser la réponse lorsqu'elle sera disponible : [http://localhost:8282/client/reponse/2]
